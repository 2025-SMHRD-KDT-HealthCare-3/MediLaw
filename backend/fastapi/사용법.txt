MediLaw DB 사용법
====================
이 zip은 medilaw.db 입니다 (의료 4법령·판례·해석례·개인정보위 결정문·보건복지부 가이드라인
+ 검색용 임베딩 벡터 포함). 재수집/재임베딩 없이 바로 검색·챗봇이 동작합니다.

[설치]
1) 이 zip을  backend/fastapi/  폴더 안에서 압축 해제 → data/medilaw.db 가 생깁니다.
2) 코드는 git(HMS 브랜치)에서 받으세요.
3) cd backend/fastapi && pip install -r requirements.txt
4) cp .env.example .env  후  OPENAI_API_KEY 를 '본인 키'로 채우세요.
     (하이브리드 의미검색·챗봇에 필요. 키 없으면 키워드(FTS) 검색만 동작)
5) 실행:  DB_PATH=data/medilaw.db uvicorn app.main:app --port 8077
     확인:  http://localhost:8077/health   →  embeddings_ready: true  이면 정상
     문서:  http://localhost:8077/docs

[주의]
- OPENAI_API_KEY 는 각자 발급(platform.openai.com), 절대 공유하지 마세요.
- 이 DB는 sqlite-vec 확장으로 검색합니다(requirements.txt에 포함).
